# grc-rpc-manager
grc-rpc-manager is a proxy made to increase GridcoinResearch's RPC capabilities by adding permissions(working but wip), caching(todo) etc.
## Installation
```
git clone https://github.com/div72/grc-rpc-manager.git
cd grc-rpc-manager
python3 main.py
```
